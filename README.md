# Physics-Informed Machine Learning

A collection of projects, tutorials and research-oriented implementations in Scientific Machine Learning, Physics-Informed Neural Networks (PINNs), Deep Operator Networks (DeepONet) and Computational Mechanics.

## Research Areas

- Scientific Machine Learning
- Physics-Informed Neural Networks (PINNs)
- Deep Operator Networks (DeepONet)
- Computational Mechanics
- Dynamical Systems Modeling
- Engineering Applications

## Repository Structure

### PINNs
Physics-informed learning for engineering and physical systems.

### Tutorials
